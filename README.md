# LTV Boost

WhatsApp Marketing SaaS para e-commerces brasileiros.

## 📖 Documentação
Para entender como utilizar a plataforma, estratégias de LTV e configurações técnicas, consulte o nosso:
👉 **[Playbook da Plataforma (PLAYBOOK.md)](./PLAYBOOK.md)**

## 🛠️ Tecnologias
- **Frontend:** React + Vite + TypeScript + Tailwind CSS
- **Backend:** Supabase
- **WhatsApp:** Meta Cloud API

## 🚀 Como começar
1. Clone o repositório
2. Instale as dependências: `npm install`
3. Configure as variáveis de ambiente: copie `.env.example` para `.env` e preencha (produção: ver `docs/production-env-checklist.md`).
4. Inicie o servidor de desenvolvimento: `npm run dev`
